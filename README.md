# IPv4/IPv6 Enterprise Network Simulation
Simulación de infraestructura de red empresarial con configuración de routers, switches, VLANs y pruebas de conectividad.

## Tecnologías
- Cisco Packet Tracer
- IPv4 / IPv6
- Routing
- Switching

## Implementaciones
- Configuración de VLANs
- Direccionamiento IPv6
- Enrutamiento
- Verificación de conectividad
